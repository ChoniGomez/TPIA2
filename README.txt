Requisitos para ejecutar la aplicación:

1- Sistema Operativo: 	Windows 10/11

2- Resoluciones compatibles: 	832 x 624
				1024 x 768
				1128 x 634
				1280 x 720
				1366 x 768

3- Escalada obligatoria:	100%

4- En caso de ejecutarlo desde el código fuente 
	4.1 Se recomienda utilizar Python 3.8
	4.2 Instalar las siguientes librerias mediante el comando "pip install":
		4.2.1 PyQt5
		4.2.2 Matplotlib
		4.2.3 Scikit-learn
